# Goal: 实现 Phase 4 学习与演化最小可用集

## One-sentence statement

让 agent 的反思笔记真正影响 Working Set 装配——通过 TF-IDF 索引检索最相关的反思，取代固定拉取最近审计行，实现"从历史中学习"的第一步。

## Lessons learned

- TF-IDF 余弦相似度用 stdlib (math, re, json) 即可实现，无需 scikit-learn 等外部依赖——满足"stdlib-only"约束。
- `_read_reflection` 需同时 catch `UnicodeDecodeError`（非法 UTF-8 文件），不只是 `OSError`。
- Layer 3 header 从 "Recent Audit" 改为 "Relevant Context" 是语义级改进，不破坏已有测试（测试只检查函数行为，不检查 header 名称）。
