✦ 功能特性

为对话而生的体验

精心设计的每一处细节，让你专注于与 AI 的对话本身。

流式实时输出

基于 Server-Sent Events 的流式输出，AI 的每一个 token 都实时呈现，无需等待完整响应。

完整 Markdown 渲染

支持代码块语法高亮、表格、列表、引用等完整 Markdown 格式，技术文档阅读体验一流。

本地 API Key 安全

API Key 仅存储在你的浏览器 localStorage 中，不经过任何第三方服务器，完全私密安全。

多模型兼容

支持 OpenAI、Azure OpenAI、DeepSeek、通义千问等任意 OpenAI API 兼容接口，自定义 Base URL。

自定义系统提示词

灵活设置 System Prompt，为 AI 指定角色、语气和专业领域，打造专属 AI 助手。

对话上下文管理

完整的多轮对话上下文，AI 记得你说过的每一句话。支持一键清空，随时开始全新对话。
